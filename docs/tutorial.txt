Hướng dẫn cài đặt và cập nhật phần mềm
# I. Những phần mềm cần thiết
1. Windows Server
	- Xampp: apache server & mysql
	- Composer: instal và update các gói vendor
	- Git: clone và pull source
	- Oracle client instant: Kết nối csdl Oracle
	- Nssm: Cài đặt windows service
	- ImageMagick: Gói xử lý qr (Sẽ bỏ trong tương lai)
	- export cấu trúc database trên bản đang chạy (qlbv.sql)
2. Cài đặt và cập nhật tool
	- Cài đặt: Sử dụng cmd chạy lệnh: 
		+ git clone https://github.com/tracnn/qlbv_public.git
		+ Xóa thư mục qlbv_public/public/storage
		+ Copy thư mục qlbv_public/storage/framework/cache ra thư mục gốc chứa phần mềm
		+ Tại thư mục gốc chứa phần mềm chạy lệnh: php artisan storage:link
	- Cài đặt các gói cần thiết: Tại thư mục chứa tool chạy lệnh: 
		composer update
	- Cập nhật tool (Định kỳ hoặc có hướng dẫn của đội triển khai)
		+ Tại thư mục chứa tool chạy lệnh: 
			git pull origin main
		+ Update cấu trúc csdl nếu có, tại thư mục chưa tool chạy lệnh: 
			php artisan migrate
3. Truy cập github tại địa chỉ: https://github.com/tracnn/qlbv_public để xem nội dung các bản cập nhật (theo tuần)
